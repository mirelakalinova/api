# REST API CAR MAKES AND MODELS


## Description
`REST API CAR MAKES AND MODELS` is REST application providing makes and model from external database

- Makes - add and delete 
- Models - add and delete 

External domains could add and delete makes and models.

## Getting Started

**Dependencies**

- Java 17
- Spring Boot 3.4.0
- Spring Data JPA / Spring Data JDBC
- MySQL 
- Spring Security
- OpenFeign
- Maven (build)
- ModelMapper
- JUnit
- Mockito
- HSQLDB
- Lombok



## Installing
### Docker image 
    Official docker image 
        > https://hub.docker.com/repository/docker/mirelakalinova/car-api
    Expample push command
        > docker push mirelakalinova/car-api:1.0

### Required environment variables
  - DB_HOST
  - DB_PORT
  - DB_USER
  - DB_PASS
  - PORT
  - ALLOWED_DOMAINS
